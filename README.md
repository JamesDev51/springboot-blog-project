# Springboot-blog-project
기본적인 CRUD 구현을 위한 블로그 만들기

## Core packages
1. Spring-boot
2. Spring-security
3. Spring-jpa
4. Thymeleaf
5. My-sql

## Features
1. SignUp/SignIn
2. CRUD Post
3. CRUD Comment
4. Edit Profile
5. Remember-me
6. Search for title or content
